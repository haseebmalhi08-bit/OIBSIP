# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project was developed as part of the **Oasis Infobyte Data Science Internship**.

The objective is to build machine learning models capable of classifying Iris flowers into three species:

- Setosa
- Versicolor
- Virginica

using four flower measurements:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## 🎯 Objective

To train and evaluate multiple machine learning classification models and identify the best-performing model for Iris flower classification.
