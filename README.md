# Unemployment Analysis with Python

## 📌 Project Overview
This project analyzes unemployment trends in India using Python. It performs Exploratory Data Analysis (EDA) to identify regional and monthly unemployment patterns and evaluates the impact of the COVID-19 pandemic on unemployment rates.

## 🎯 Objective
- Analyze unemployment data in India.
- Explore region-wise and month-wise unemployment trends.
- Compare unemployment rates before and after COVID-19.
- Visualize insights using charts and graphs.

## 🛠️ Technologies Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

## 📊 Features
- Data Loading and Inspection
- Data Cleaning
- Missing Value Analysis
- Region-wise Average Unemployment Rate
- Month-wise Trend Analysis
- Time Series Analysis for Major States
- Top 10 States with Highest Unemployment Rate
- Correlation Heatmap
- Pre-COVID vs Post-COVID Comparison
- Visualizations with Observations

## 📁 Dataset
- **Dataset Name:** Unemployment in India
- **Source:** Kaggle

## 📈 Output
The notebook generates:
- Region-wise bar charts
- Monthly trend line chart
- Time-series analysis
- Top 10 states bar chart
- Correlation heatmap
- COVID-19 impact comparison
